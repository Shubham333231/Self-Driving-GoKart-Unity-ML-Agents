# Self-Driving Go-Kart using Unity ML-Agents

A self-driving Go-Kart simulation trained using Reinforcement Learning 
via Unity ML-Agents framework.

## Tech Stack
- Unity3D
- Unity ML-Agents (PPO Algorithm)
- Python
- C#
- Reinforcement Learning

## Overview
Trained an autonomous agent to navigate a Go-Kart track without human 
input using Proximal Policy Optimization (PPO). Reward engineering used 
to shape agent behavior — positive rewards for track progression, 
penalties for collisions.

## Reference
Based on architecture by Fabrizio Frigeni.
