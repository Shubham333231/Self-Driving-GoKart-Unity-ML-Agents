A self-driving Go-Kart simulation trained using Reinforcement 
Learning via Unity ML-Agents framework using Proximal Policy 
Optimization (PPO) algorithm.

## 🎯 Project Overview
Trained an autonomous agent to navigate a Go-Kart track 
without any human input. The agent learns entirely through 
trial and error using reward signals.

## 🧠 ML Concepts Used
- Reinforcement Learning (RL)
- Proximal Policy Optimization (PPO)
- Reward Engineering
- Neural Network Policy Training
- Exploration vs Exploitation

## ⚙️ Tech Stack
| Technology | Purpose |
|-----------|---------|
| Unity3D | Simulation Environment |
| Unity ML-Agents | RL Training Framework |
